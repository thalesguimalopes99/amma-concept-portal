Instruções de Uso

Esta assinatura usa estilos inline para compatibilidade total com clientes de e-mail. Não depende de fontes externas, usa fallbacks seguros.

Gmail: Configurações → Ver todas as configurações → Assinatura → Criar nova → clique no ícone </> (HTML) → cole o código abaixo.

Outlook: Arquivo → Opções → Email → Assinaturas → Nova → cole em modo HTML.

Substitua [SEU NOME] e [CARGO] antes de aplicar. Opções de cargo: Fundadora · Artista de Beleza · Atelier de Beleza.
